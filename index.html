Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • all tags are contained inside of angle brackets
  • H1 - H6 represent different levels of heading
  • P represnts a paragraph
  • img contains information the are displayed on the webpage
  • Br creates a line or empty space
  • A creates a link to the same webpage or other webpages. Href attributes points to the URL for the link
  • UL represents unordered list
  • OL represent ordered list
  • LI represents listed item

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Card designates something as a class selectior
  • box-shadow adds effects around an element's frame

GIT

  • Git status: checks what branch we are currntly on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: to return to your main branch
  • git pull origin main: use this to make sure you are working on the most up to date virsion of the project

JavaScript

  • >A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
